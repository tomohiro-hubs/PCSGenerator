# Excel Transfer & PCS Daily Generator

月次データ（Excel）からPCSデータを抽出し、日次差分を計算してテンプレート（Excel）へ転記するWebアプリケーションです。
ブラウザ上で全ての処理が完結するため、データが外部サーバーに送信されることはありません。

## 主な機能

1. **Excelデータ変換**
   - 月次レポート（累積値）から日次差分（当日 - 前日）を自動計算
   - テンプレートファイルへの自動転記
   - PCS列（553列など）の自動生成とスタイルコピー
   - 日射量データの転記

2. **検証機能**
   - 必須ヘッダの存在確認
   - 日付の連続性チェック
   - 負の値や欠損データの検出（警告として表示）

3. **エラーレポート**
   - 生成されたExcelファイルに「errors」シートを自動追加
   - アプリケーション画面上でもエラー/警告を検索・フィルタ可能

4. **設定カスタマイズ**
   - シート名、ヘッダ行番号、期待するPCS数などを設定画面で変更可能
   - 設定はブラウザ（Local Storage）に保存

## 使用方法

1. **変換タブ**
   - 「月次データ」と「テンプレート」のExcelファイルを選択します。
   - 「検証 (Validate)」ボタンをクリックします。
   - エラーがなければ「生成・ダウンロード (Generate)」ボタンが有効になります。
   - 生成された `filled_template.xlsx` を保存します。

2. **エラータブ**
   - 検証時に発生したエラーや警告の詳細を確認できます。

3. **設定タブ**
   - Excelのシート名や行番号のルールを変更できます。
   - 初期設定に戻すことも可能です。

## 技術スタック

- **Frontend**: React, TypeScript, Vite
- **UI**: Tailwind CSS, Lucide React
- **Excel Processing**: ExcelJS
- **Routing**: React Router DOM (HashRouter)

## 開発・デプロイ

GitHub Pagesでのホスティングを想定した完全静的アプリケーションです。

### ローカル開発

```bash
npm install
npm run dev
```

### ビルド

```bash
npm run build
```

`dist` フォルダに出力されたファイルをWebサーバーに配置してください。

## データプライバシー

本アプリケーションはクライアントサイド（ブラウザ）でのみ動作します。アップロードされたExcelファイルや解析データが外部のサーバーに送信・保存されることは一切ありません。
